# @computesdk/tensorlake

## 0.1.0

### Minor Changes

- b4ad62c: Add `@computesdk/tensorlake` provider for stateful MicroVM sandboxes powered by Tensorlake (https://tensorlake.ai), aimed at agentic applications and LLM-generated code execution.
