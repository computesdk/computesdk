---
"@computesdk/tensorlake": minor
---

Add `@computesdk/tensorlake` provider for stateful MicroVM sandboxes powered by Tensorlake (https://tensorlake.ai), aimed at agentic applications and LLM-generated code execution. Wraps the `tensorlake` SDK and is auto-detected by the `computesdk` gateway via the `TENSORLAKE_API_KEY` environment variable.
